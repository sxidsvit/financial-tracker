##  Financial tracker App


#### 🛠️ Stack 
- Nextjs 14: API routes and ServerActions
- Auth with Clerk
- Typescript
- Tailwindcss / Shadcn UI
- SQLite / Vercel PostgreSQL
- Prisma as ORM
- React-query
- Recharts

#### 📗 Features
- Transaction Management
- Category Statistics
- Historical Data Aggregates
- Stunning Charts
- Icon Selector for Categories
